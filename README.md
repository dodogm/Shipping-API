# 📦 Shipping API
### API em Node.js para gerenciamento de entregas


---

## 📖 Sobre o Projeto
O **Shipping API** é uma aplicação backend desenvolvida em **Node.js** que fornece endpoints para gerenciar envios e entregas.  
O objetivo é praticar conceitos de APIs REST, manipulação de dados e integração com sistemas externos.

### 🛠️ Tecnologias Utilizadas
- Node.js
- Express.js
- JavaScript
- Git & GitHub
- Prisma
- Bcrypt
- Jsonwebtoken
- Zod

---

## ⚙️ Guia de Instalação e Execução

### 📌 Pré-requisitos
- **Node.js** instalado (versão 18 ou superior recomendada)
- **npm** ou **yarn**

### 🚀 Passo a passo
```bash
# Clonar o repositório
git clone https://github.com/dodogm/Shipping-API.git

# Entrar na pasta do projeto
cd Shipping-API

# Instalar dependências
npm install

# Rodar a aplicação
npm run dev
